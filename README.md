### AmiTemplateのカスタマイズ版
[本家AmiTemplate](https://github.com/amishiro/AmiTemplate-PHP)のv2.0.20をベースに
自分用にカスタマイズしています。



### QUICK START

１）ローカルサーバー立ち上げとnpmパッケージインストール

```
$ vagrant up
```
&
```
$ npm i
```

２）gulpタスクスタート

```
$ npm run gulp
```

---

### 動作確認ずみバージョン

